# Logo Assets

This directory contains the brand logos for The Concurrent Contractor™ application.

## Required Logo Files

Please place the following logo files in this directory:

1. **tcc-logo.png** - The gold TCC icon logo
   - Use: Primary brand icon, header logo
   - Recommended size: 200x200px or larger (transparent background)

2. **tcc-text-logo-black.png** - "The Concurrent Contractor™" text logo in black serif font
   - Use: Alternative header logo, reports, documents
   - Recommended: Transparent background

3. **tcc-text-logo-framed.png** - "The Concurrent Contractor™" text logo with gold border
   - Use: Report headers, presentations, formal documents
   - Recommended: White or transparent background

## Brand Colors (from logos)

- **Primary Gold**: #D4AF37 (darker gold) to #FFD700 (bright gold) - gradient effect
- **Secondary Orange**: #FF8C00
- **Text Black**: #000000
- **Background**: #1A1A1A (dark) / #000000 (black)

## Usage Notes

- The TCC icon logo uses a gold gradient effect
- Maintain aspect ratios when resizing
- Use high-resolution versions for print materials
- Ensure sufficient contrast against dark backgrounds
